# CSRA Application
Cell Sharing Risk Assessment application.

## Install dependencies
```
yarn
```

## Build
```
yarn run build
```

## Test
```
yarn test
```

## Run
```
yarn start
```

Then go to [http://localhost:8080/](http://localhost:8080/)

## Run in development mode
```
yarn run start:dev
```

Then go to [http://localhost:3000/](http://localhost:3000/)
